# -Estudo-de-Caso-Agilidade-no-Setor-Bancario
Estudo comparativo de agilidade no setor bancário: a transição do Banco Daycoval para o modelo de Squads e Kanban vs. o DNA digital do Nubank. Analiso como o Daycoval aplica Agilidade Pragmática para modernizar o crédito, enquanto o Nubank escala via Contínuos Deployment e autonomia, superando rituais rígidos de Scrum.
